# 🔐 Password Generator

A responsive Password Generator built using HTML, CSS, and JavaScript.

## 🚀 Features
- Random secure password generation
- Password length customization
- Symbols and numbers support
- Uppercase/lowercase options
- Copy to clipboard functionality

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## 📂 Project Flow
1. Created character sets
2. Added password length control
3. Implemented random password logic
4. Added checkbox option handling
5. Implemented copy button functionality
